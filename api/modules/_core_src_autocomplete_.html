Accept

Accept: (item: Item) => string | undefined
  • Defined in core/src/autocomplete.ts:65

Accepts the specified autocomplete item.

param

The item to accept.

returns

If undefined is returned, Autocomplete will do nothing.

If a string is returned, Autocomplete will:

  • use it as a replacement for the currently matched text
  • clear autocomplete state
  • focus the editor
